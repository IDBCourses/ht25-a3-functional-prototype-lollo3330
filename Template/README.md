// MAZE GAME OVERVIEW //

This is a interactive maze game built by HTML, CSS and JavaScript. The player navigates through a grid based maze using keyboard controls.

// GOAL OF GAME //
Navigate the player through the maze and reach the goal without stepping on any traps and getting to the goal before the timer runs out.

// PLAYER MOVEMENT //
Movement through the maze is made with a simple key click in a direction.
Hold down a key to move quicker in that direction.

MOVE UP = Top row keys (QWERTYUIOP)
MOVE DOWN = Bottom row keys (ZXCVBNM)
MOVE LEFT = Middle-left keys (ASDFG)
MOVE RIGHT = Middle-right keys (HJKL)

// TIMER LOGIC //
Each round starts with a countdown timer. You must reach the goal before time runs out. If the timer hits zero, you lose the round.

// RESETTING THE GAME //
Press the reset button to start a new round.
